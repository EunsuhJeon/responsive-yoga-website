# Responsive Yoga Website

## Overview
This project consist of a **web page** about **Yoga classes**.

## Objectives
Learn and practice creating a long single-page landing site that is responsive for mobile and desktop, toggles between two themes (dark and light), including animations and transitions.

## Languages
- **HTML**: for semantic structure and accessibility.  
- **CSS**: for custom layout design and styling.
